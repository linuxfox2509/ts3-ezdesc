# ts3-ezdesc

Tool for TS3 Users to make channel descriptions without having to type the BBCode yourself.

To build for Windows run: npm run build
To build for Linux run: npx electron-packager . ts3-ezdesc --platform=linux --arch=x64
To build for macOS run: npx electron-packager . ts3-ezdesc --platform=darwin --arch=x64

Change window size by modifying height and width in main.js
